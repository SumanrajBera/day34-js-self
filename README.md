# Events and Event Handling
- Events are interaction that user do with the browser. Ex: clicking, typing, etc.
- Event Handling is the process of performing some action based on the events.

## Basic Events
- click
- dblclick
- change
- input
- keydown
- mouseover and mouseout

## Features and Techniques
- Prevent Default
- Event Delegation
- Event Bubbling
- Event Capturing
